Git教程

发表于2022-10-27|更新于2022-10-27
|阅读量:

1、在任意目录下右键–“Git Bash Here”

2、命令行内输入 git config –global user.name “username”

git config –global user.email”useremail”

git config –list:查看所有全局配置信息

3、获取git仓库

1)本地初始化

进入任意空目录：

git init

2)从远程仓库克隆（常用）

复制远程仓库地址

git clone + 地址

工作区 —-> git add * —->暂存区 —->git commit —->版本库

工作区文件的状态

1）untracked 未被纳入版本控制

2）tracked 已被纳入版本控制

​ a.Unmodified 未被修改

​ b.Modified 已被修改

​ c.Staged 已暂存

git本地常用命令

git status 查看文件状态

git add 将文件的修改加入暂存区

git reset 将暂存去的文件取消暂存或者是切换到指定版本

git commit 将暂存区的文件修改提交到版本库

git log 查看日志

git远程仓库常用命令

git remote 查看远程仓库

git remote add 添加远程仓库

git clone 从远程仓库克隆

git pull 从远程仓库拉去

git push 推送到远程仓库
